Hexo的搭建(hexo 3.0)

初衷

这几天，准备搭建一个自己的博客，看到hexo的强大搭建博客能力和容易上手，用hexo搭建了自己的博客，看到网上使用hexo搭建博客的教程很多，我自己也是跟着各位大神的教程搭建的博客，写这篇博客是为了记录下我在搭建的过程中遇到的问题和个人学习笔记。

安装Git

下载安装msysgit

安装Node.js

在 Windows 环境下安装 Node.js ，下载Node.js文件安装即可，结果后面使用命令安装hexo的时候，提示无效的命令，可以重启下机器。

这要提下安装hexo的时候可能会没有安装的light的主题，到Github克隆light主题

把克隆的light主题文件夹light，复制到F:\hexo\themes下，这个时候在访问wuliangchen.github.io就可以了。

hexo的配置文件

hexo里面有两个常用到的配置文件，分别是整个博客的配置文件F:\hexo_config.yml和主题的配置文件F:\hexo\themes\light_config.yml。本博客使用的主题是基于light改善的主题。如果你想自己挑选喜欢的主题，hexo官方提供主题供你自己选择，使用方法很简单，点击自己想要的主题，进入该主题的Repository，使用Git将主题clone到本地，然后将整个文件夹复制到F:\hexo\themes文件夹下，将F:\hexo_config.yml里的theme名字改为自己下载的主题的文件夹名即可。

配置整个博客的文件E:\hexo_config.yml

运行hexo g，hexo s，打开localhost:4000浏览效果。

配置博客主题的文件F:\hexo\themes\light_config.yml

发表一篇新文章

1.在Git Bash执行命令：$ hexo new “my new post”
2.在F:\hexo\source_post中打开my-new-post.md，打开方式使用自己常用的编辑器就行。

写完文章后，你可以使用
1.$ hexo g 生成静态文件。
2.$ hexo s 访问localhost:4000预览效果。
3.$ hexo d 同步到github，然后使用http://wuliangchen.github.io进行访问。

hexo中写文章使用的是Markdown。
